# Practica-Sumativa-1
Este proyecto es una práctica introductoria en PHP donde se desarrollan distintas operaciones aritméticas básicas utilizando variables y operadores. Los resultados se muestran directamente en pantalla mediante el uso de echo.

Objetivo de la práctica

El propósito de esta práctica es reforzar los siguientes conceptos fundamentales:

- Uso de variables en PHP
- Aplicación de operadores aritméticos
- Impresión de resultados en pantalla
- Comprensión del orden de precedencia en las operaciones matemáticas
Este repositorio contiene varios ejemplos prácticos en PHP para aprender y reforzar conceptos fundamentales, tales como:

🔹 Variables
🔹 Arreglos
🔹 Funciones

🌐 Tecnologías utilizadas
PHP Apache MySQL XAMPP WampServer
